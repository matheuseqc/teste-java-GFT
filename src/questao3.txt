Crie uma classe em Java chamada Projeto para gerenciar um projeto chamado "Aplicativo Mobile", com orçamento de R$5000,00 e 4 membros na equipe. A classe deve permitir acessar e alterar os dados do projeto, garantindo que orçamento e o número de membros sejam sempre positivos. Inclua um método que calcule o orçamento médio por membro. Escreva comentários explicando o código e siga boas práticas de programação orientada a objetos. Também crie uma classe principal que crie o objeto Projeto e mostre como usar os métodos.
